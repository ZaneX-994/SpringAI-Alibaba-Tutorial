### Spring AI Alibaba 教程项目

#### 项目简介

本教程代码源自尚硅谷的 Spring AI Alibaba 网课教学内容，是一个完整的 Spring AI 集成阿里云通义千问大模型的实战学习项目。

#### 项目结构
````
SpringAIAlibaba-Tutorial/               # 父工程
├── tutorial-01-HelloWorld/             # 基础示例：Spring AI 快速入门
├── tutorial-02-ollama/
├── tutorial-03-ChatModelChatClient/
├── tutorial-04-StreamingOutput/
├── tutorial-05-ChatPrompt/
├── tutorial-06-PromptTemplate/
├── 后续添加更多模块...
└── README.md                           # 项目说明文档
````

#### 🎯 实战案例：AIWorkHelper 企业办公助手
AIWorkHelper是基于Java和Spring AI Alibaba开发的AI驱动企业办公助手平台，集成阿里云通义千问大模型，通过Agent自主编排和Function Calling机制，为企业提供代办管理、审批流程、知识库检索等业务的智能化处理服务。
